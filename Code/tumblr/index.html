房総の旬の魚や野菜、果物を使った “心と体にやさしい料理” を提供しています。冷たいドリンクやかき氷などは自作のガラス器でお出ししています。

入口の扉を開けると、左右に並べられた色とりどりのガラス作品がお客様をお出迎えします。カフェにあがるまでの空間はショップコーナーになっています。

玄関から一段上がると、フローリングのテーブル席
（4席）に加え、奥には畳のお座敷席（4席と2席）があり、足をのばしてくつろげます。赤ちゃん用に布団もありますので「ねんね」のお子様がいても安心してお越しいただけます。

天気が良い日は、テラス席もおすすめです。目の前に広がる澄んだ海と木漏れ日の中で、ティータイムをお楽しみいただけます。（ペットも同伴できます）

ガラス工房では様々なガラス工芸制作の体験が出来ます。

宙吹きガラス体験

約1,300度に溶けたガラスを金属製の吹き竿に巻き取り、息を吹き込みます。次に「ジャック」という大きなピンセットのような道具でガラスの形を整えていきます。
コップや一輪挿しといった生活で使える様々なガラス器を制作することができます。お子様からお年寄りまで、皆様で挑戦してみてください。

  • 制作時間：   30分前後
  • 料金：   １点 3,500円（税別）
  • 制作できるもの：   コップ、一輪挿し、ボールなど
  • 高温場所での作業になります。タオルや飲み物をご用意ください。
    服装は肌が隠れ、動きやすく汚れてもよい格好でご参加ください。
    綿100％の素材が安全です。必ず運動靴をご着用ください。
    （化学繊維、短パン、サンダルなどは火傷や怪我の原因になります）
  • 制作した作品のお渡しは翌日以降になります。配送の場合は着払いでの発送になります。

サンドキャスト —お子様の手型・足型に—

水で湿らせた砂に手足を押し当て、できたくぼみに溶けたガラスを流し込んで作品を作ります。お子様の成長の記念にいかがですか？（お名前や日付を刻み込む事もできます）

  • 制作時間：   15分前後
  • 料金：   1点 3,500円（税別）
  • 制作できるもの：   お子様の手形・足型が入ったガラス置物
  • 制作した作品のお渡しは翌日以降になります。配送の場合は着払いでの発送になります。

サンドブラスト

サンドブラスト体験では、まずご自分のオリジナルデザイン画を器に貼ったシールへ転写し、そのデザイン画をカッターで切り抜きます。つぎに、切り抜いた部分に「サンドブラスター」という道具を使い、砂を含んだ圧縮空気を吹き付けガラスの表面をくもりガラスに仕上げることで、模様を施すことができます。

  • 制作時間：   40分〜1時間30分
